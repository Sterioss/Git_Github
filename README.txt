# Git_Github
Projet a but pédagogique. Apprentissage des fonctions de Git et de Github via la pratique.
Le cours se déroule sur un site web dédié à l'apprentissage en ligne ; Openclassrooms.com.

Le projet contient deux fichier tiré de "placeholders" afin d'obtenir rapidement du remplissage texte dans ses projets.
